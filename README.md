# 💰 Expense Tracker

A simple and responsive Expense Tracker web application built using **HTML**, **CSS**, and **JavaScript**. It helps users manage their income and expenses by tracking transactions and automatically calculating the current balance.

---

## 📌 Features

- ➕ Add Income
- ➖ Add Expense
- 💰 Calculate Total Balance
- 📈 Calculate Total Income
- 📉 Calculate Total Expense
- 🗑️ Delete Transactions
- 💾 Save Data using Local Storage
- 🔄 Automatically Load Data after Refresh
- 📱 Responsive User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## 💡 How It Works

1. Enter a description.
2. Enter the amount.
3. Select Income or Expense.
4. Click **Add Transaction**.
5. View updated Balance, Income, and Expense.
6. Delete any transaction if needed.
7. Data is automatically saved using Local Storage.
